## Subject
### Demand

本课程的实验课时为4周，现在按照6周的时间来准备，进度安排如下：

1）第1周为准备期，完成组队和选题，建立起代码库（GitHub）和博客展示（cnblog或其他）；

2）第2周为需求捕获和分析，网上提交相应的文档；

3）第三周为系统设计，网上提交相应的文档；

4）第四周为编码实现及单元测试，代码上传至GitHub；5）第五周为系统集成，网上提交相应的文档；

6）第六周为机动时间，查漏补缺；

7）14周末进行验收。评选3个优秀项目进行奖励
